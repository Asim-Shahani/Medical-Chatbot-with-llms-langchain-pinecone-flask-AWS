# Medical-Chatbot-with-llms-langchain-pinecone-flask-AWS

# how to run?

### Steps:

Clone the respistory
```bash
git clone  https://github.com/Asim-Shahani/Medical-Chatbot-with-llms-langchain-pinecone-flask-AWS.git
```
### STEP 01- Create a conda environment after opening the respository 
```bash
conda create -n medibot python-3.11 -y
```

```bash
conda activate medibot
```

### STEP 02- install the requirements
```bash
pip install -r requirements.txt
```
